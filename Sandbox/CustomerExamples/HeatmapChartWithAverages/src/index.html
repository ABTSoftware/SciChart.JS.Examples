Heatmap Chart with Averages

Selecting Points

  1. Open Chrome DevTools Console tab
  2. Uncheck Heatmap 2 checkbox, to hide the second Heatmap chart
  3. Select area on the chart
  4. In console you will get the information about start and end indexes.
    Bear in mind that you need also to check if the start and the end indexes are in the available range for ZValues
  5. Pan the chart holding the right mouse key
